# 🌧️ 구구단 산성비 게임 (Multiplication Rain)

초등학교 2학년 학생들이 곱셈구구를 재미있게 학습할 수 있는 게임입니다.

## 🎮 게임 소개

하늘에서 곱셈 문제가 적힌 물방울이 떨어집니다. 
바닥에 닿기 전에 정답을 입력하여 물방울을 터뜨리세요!

### 주요 기능

- **3가지 게임 모드**
  - 📚 연습 모드: 원하는 단을 선택해서 집중 연습
  - 🎯 스테이지 모드: 2단부터 9단까지 순차 도전
  - 🔥 챌린지 모드: 랜덤 문제로 최고 점수 도전

- **학습 지원**
  - 실시간 피드백 시스템
  - 콤보 시스템으로 동기 부여
  - 단별 학습 통계 제공
  - 배지 시스템

- **편리한 설정**
  - 사운드/음악 조절
  - 키보드/버튼 입력 방식 선택
  - 난이도 조절
  - 힌트 표시 옵션

## 🚀 시작하기

### 필요 사항

- Node.js (v16 이상)

## 🎯 게임 방법

1. **메인 메뉴에서 모드 선택**
   - 연습 모드: 2단~9단 중 선택
   - 스테이지 모드: 2단부터 시작
   - 챌린지 모드: 랜덤 문제

2. **난이도 선택** (연습/스테이지 모드)
   - 😊 쉬움: 천천히, 물방울 적음
   - 😐 보통: 중간 속도
   - 😤 어려움: 빠른 속도, 물방울 많음

3. **게임 플레이**
   - 물방울에 적힌 곱셈 문제 확인
   - 키보드 숫자 키 또는 화면 버튼으로 정답 입력
   - Enter 키 또는 ✓ 버튼으로 제출
   - 연속 정답으로 콤보 달성!

4. **점수 시스템**
   - 기본 점수: 정답 1개당 10점
   - 콤보 보너스: 5/10/15 연속 정답 시 추가 점수
   - 속도 보너스: 빠르게 답할수록 추가 점수
   - 완벽한 스테이지: +200점

## 🎨 주요 기능 상세

### 게임 모드

1. **연습 모드 (Practice Mode)**
   - 특정 단 집중 학습
   - 제한 시간 없음
   - 단별 최고 점수 저장

2. **스테이지 모드 (Stage Mode)**
   - 2단부터 순차 진행
   - 각 단마다 10문제 클리어 필요
   - 단계별 난이도 상승

3. **챌린지 모드 (Challenge Mode)**
   - 2~9단 랜덤 출제
   - 높은 난이도
   - 최고 점수 경쟁

### 학습 통계

- 전체 게임 수, 정답 수, 정확도
- 모드별 최고 점수
- 단별 연습 모드 점수
- 획득한 배지 표시

### 배지 시스템

- 🌱 구구단 초보: 첫 스테이지 클리어
- 🥉/🥈/🥇 단 마스터: 각 단 30문제 연속 정답
- 🏆 구구단 천재: 모든 단 마스터
- 💎 완벽주의자: 무실수 클리어
- 🔥 집중력 왕: 50콤보 달성

## 🎓 교육적 효과

1. **반복 학습**: 게임을 통한 자연스러운 구구단 반복
2. **즉각 피드백**: 정답/오답 즉시 확인
3. **동기 부여**: 점수, 콤보, 배지 시스템
4. **난이도 조절**: 학습자 수준에 맞는 선택
5. **진행 추적**: 통계로 학습 상태 파악
